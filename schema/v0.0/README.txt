# HEMS SVA JSON Starter

含まれるファイル:
- sva-design.schema.json
- usecase.schema.json
- nfr-risk.schema.json
- ipower-engine.sva.instance.json
- ipower-engine.usecase.instance.json
- ipower-engine.nfr-risk.instance.json

使い方:
1. schema を設計治具として固定
2. instance json に案件内容を入れる
3. instance json から md を自動生成
4. md を Word に変換して客先提出物にする

次の推奨ステップ:
- md テンプレート作成
- docx テンプレート作成
- JSON -> Markdown 変換スクリプト作成
